About the Project

This project demonstrates a static website setup using Nginx on an AWS EC2 instance. It includes setting up a domain, configuring SSL with Let's Encrypt, and automating deployments with GitHub Actions.

Features

  • Automated Deployment with GitHub Actions
  • Secure SSL Encryption with Let's Encrypt
  • Scalable Web Hosting with AWS EC2
  • Custom Domain Configuration
